## ğŸ“– Usage Guide

### 1. Personal Information
- Enter your **Full Name** (required)
- Add your **Career Objective** (optional)
- Upload a **Profile Picture** (recommended)

### 2. Contact Details
- **Email Address** (required, validated)
- **Phone Number** (required, validated)
- **Address** (optional)

### 3. Work Experience
Fill in the following for each position:
- Job Title
- Organization Name
- Duration (e.g., "Jan 2020 - Present")
- Job Description

Click **"+ Add Experience"** to add multiple positions.

### 4. Education
Add your educational qualifications:
- Degree/Qualification
- Institution Name
- Year/Duration
- Result/GPA

Click **"+ Add Education"** to add multiple entries.

### 5. Skills
- Enter relevant skills
- Add as many skills as needed
- Each skill appears as a separate item in your CV

### 6. Languages
- Select languages you speak
- Choose proficiency level:
  - Native
  - Fluent
  - Advanced
  - Intermediate
  - Basic

### 7. References
Add professional references:
- Name
- Designation
- Organization
- Email
- Phone

### 8. Download PDF
- Review your CV in the live preview
- Click **"ğŸ“¥ Download PDF"** button
- Your CV will be downloaded as a PDF file
